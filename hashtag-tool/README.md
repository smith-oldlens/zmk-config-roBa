# ハッシュタグローテーター

Instagram 用のハッシュタグを毎回自動で組み替えて生成・コピーできる、単一 HTML のツールです。
サーバー不要・`index.html` をブラウザで開くだけで動きます(スマホ / PC 両対応)。

## できること

- **タグセットの自動生成** — 登録したタグからビッグ約2割 / ミドル約5割 / ニッチ約3割の配合でランダムに選出。前回使ったタグは選ばれにくくなり、同一セットの繰り返しを避けられます
- **検索・並び替え** — 名前順 / 投稿規模順 / 使用回数順、カテゴリ絞り込み
- **ワンタップコピー** — スペース区切り / 改行区切りを選択可。コピーすると履歴と使用回数に記録
- **生成履歴** — 直近20件を保存、前回との重複率を表示
- **端末間のデータ共有** — 「書き出し(コピー)」したテキストを別端末で「貼り付けて取り込み」

データはブラウザの localStorage に保存されます。

## プリセット

幻想的なオールドレンズ・ポートレート作風(白ドレスの人物 × 鹿児島の自然、プロジェクションレンズのぐるぐるボケ)に合わせて、以下6カテゴリを初期登録済みです。

- **基本** — 写真好きコミュニティ系
- **オールドレンズ** — #オールドレンズ #プロジェクションレンズ #ぐるぐるボケ #apollon など
- **幻想・世界観** — #幻想的な写真 #世界観のある写真 #物語のある写真 など
- **ポートレート・被写体** — #ポートレート #被写体 #ポトレのセカイ など
- **自然・花・鹿児島** — #鹿児島 #桜 #菜の花 #森の中 など
- **掲載・リポスト狙い** — #tokyocameraclub #genic_mag #nicostop などメディアのフィーチャータグ

「タグ管理」から自由に追加・削除できます。使用機材のレンズ名(例: 手持ちのプロジェクションレンズ名)を追加しておくと、毎回自動で混ざります。

## 使い方

1. `index.html` をブラウザで開く(スマホはファイルを共有するか、後述のGitHub Pagesを利用)
2. 「生成の設定」で自分のジャンルのカテゴリにチェック
3. 「タグセットを生成」→「コピー」→ Instagram に貼り付け

## GitHub Pages で公開する場合

リポジトリの Settings → Pages → Branch を `main`(/root)にすると、
`https://<ユーザー名>.github.io/<リポジトリ名>/hashtag-tool/` でスマホ・PCどちらからも開けます。
