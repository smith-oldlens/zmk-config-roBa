# 05. 製品化ロードマップ(将来の販売に向けて)

構想: 本ツールを「Sony 機対応のスポーツ写真自動セレクト製品」として販売する
(プリセット販売のようなデジタル商材。まず Sony 機、まず少年野球/スポーツ用途)。
本ドキュメントは 2026-07 時点のライセンス・市場調査に基づく。

## 2. 配布アーキテクチャ(製品版)

- **推論はすべて ONNX Runtime(CPU)**。torch は配布物に含めない
  → ランタイム 10〜30MB + モデル数十 MB、インストーラ 100〜300MB 級
- **パーソナライズ学習も顧客 PC 内で完結**: SigLIP2(ONNX)埋め込み + scikit-learn の
  線形分類器のみ。顧客の .lrcat を読んで「その人の選び方」を学習(spec §10 と同じ機構)
- **汎用モデル同梱**: 開発者の教師データ+協力者データで作った「野球/スポーツセレクト」の
  事前学習分類器を同梱し、パーソナライズは差分(上書き)として保存
- **FTP サーバ内蔵**(pyftpdlib): FileZilla の手動設定を排し、アプリ内ウィザードで
  「カメラに入力する IP/ユーザー/パスワードを画面に表示 → 接続テスト」まで誘導
- **ローカル完結が中核価値**: 子供の写真をクラウドに送らない。通知は数値のみ(docs/01 §6)

## 3. カメラプロファイル(Sony 機種対応の枠組み)

機種差はコードでなくデータで持つ(`profiles/sony_*.yaml`)。プロファイル項目:
- `af_tags`: AF 座標タグの優先順(FocusLocation / FocusLocation2 / FlexibleSpotPosition)
- `af_center_fallback`: FocusLocation が中心座標を返す仕様への対応(spec §7.2)
- `ftp`: FTP background transfer の有無 / 5GHz 対応 / 自動転送設定の手順書 ID

ExifTool Sony.pm の実地調査に基づく世代区分:
| 世代 | 代表機種 | AF 座標 | 備考 |
|---|---|---|---|
| 2015〜 | α7R II 以降のほぼ全機種 | FocusLocation (0x2027) | 「W H X Y」左上原点。**Sony 横断対応の基盤** |
| 2020〜 | α7S III / α1 / α7 IV 以降 | + FocusFrameSize (0x2037) | AF 枠サイズも取れる |
| 2023〜 | α9 III 以降 | + FocusLocation2 (0x204a) | FocusLocation と同値 |

FTP 対応(無線テザー可)は α7 III 以降のフルサイズ+α9/α1 系。APS-C(α6400/α6700 等)は
カード運用モードで対応(パイプラインは共通なので「対応機種」として販売可能)。

## 4. ロードマップ

| 段階 | 内容 | 完了条件 |
|---|---|---|
| **P1** | 個人版完成 = M0〜M6(docs/03)。自分の実戦で価値実証 | 1 シーズン運用、目視枚数 1/5 以下 |
| **P2** | 製品化基盤: 汎用分類器の作成(協力者データ収集)/ カメラプロファイル外部化 / FTP サーバ内蔵+ウィザード / third-party-licenses 整備 | 自分以外の 2〜3 人(チームの他の保護者)がセットアップ手順書だけで使える |
| **P3** | GUI(進捗表示・設定・パーソナライズ実行)+ PyInstaller インストーラ + ライセンスキー(Paddle/Lemon Squeezy の発行機構) | クローズドβ 10 人 |
| **P4** | 販売開始: 国内 BOOTH(手数料 5.6%+45円)+ 海外 Paddle or Lemon Squeezy(MoR 型で VAT 代行) | — |

## 5. 価格ポジション(2026-07 相場)

- 商用競合: Aftershoot $10〜45/月、Narrative Select $10〜60/月(いずれもウェディング/
  ポートレート最適化。**スポーツ・少年野球特化+ローカル完結は空白ポジション**)
- 個人開発品の現実的レンジ: **買い切り ¥5,000〜15,000** または $5〜10/月。
  まず買い切り(BOOTH 文化圏と相性が良く、決済/解約管理が不要)から始めるのを推奨
- 無料版(枚数制限 or カード運用のみ)→ 有料版(無制限+FTP+パーソナライズ)の
  フリーミアム構成は P4 で判断

## 6. リスク・留意点

- 「Sony 対応」を謳う場合、実機検証できない機種はプロファイルを「未検証」と明示する
  (FocusLocation の世代共通性が根拠でも、販売物としては検証済みリストで謳う)
- Adobe/Sony の商標: 「Lightroom 対応」「Sony 対応」は相互運用性の記述として可だが、
  ロゴ使用・公認を示唆する表現は不可。製品名にも含めない
- 顧客の .lrcat を読む機能は「読み取り専用・コピーに対して実行」を仕様として明記(spec §10.1 と同じ)
- 子供の写真を扱う製品なので、プライバシーポリシーで「画像は端末外に送信しない」を明文化
  (これが競合クラウド型に対する差別化の核でもある)
